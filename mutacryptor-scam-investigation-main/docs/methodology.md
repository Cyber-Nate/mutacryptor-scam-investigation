# Methodology

The investigation followed a structured threat intelligence approach:

## 1. Data Collection
- Emails from the organization
- Employee portal screenshots
- Payment interface analysis

## 2. Infrastructure Analysis
- Domain resolution checks
- DNS lookup failures (NXDOMAIN)
- Hosting observations

## 3. Behavioral Analysis
- Communication patterns
- Task requirements
- Payment pressure tactics

## 4. Financial Analysis
- Bitcoin wallet tracking
- Payment flow structure
- Transaction characteristics

## 5. Correlation
- Comparison with known scam patterns
- Identification of repeated tactics

## 6. Documentation
- IOC extraction
- Attack chain modeling
- Report compilation
