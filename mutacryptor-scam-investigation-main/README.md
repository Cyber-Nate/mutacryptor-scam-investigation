# 🚨 MutaCryptor Scam Network Investigation

A real-world cybersecurity investigation into a coordinated fake internship scam leveraging social engineering, fake company infrastructure, and cryptocurrency payments.

## 🔍 Key Findings
- Fake companies used as recruitment funnels
- Forced software purchase via Bitcoin-only payment
- Rotating crypto wallets to avoid tracking
- Infrastructure disappearance (NXDOMAIN)
- Multi-entity correlation (Zorvyn, VRV Security, MutaEngine)

## ⚠️ Impact
Targets entry-level cybersecurity professionals globally.
